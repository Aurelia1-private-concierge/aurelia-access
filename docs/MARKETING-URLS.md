## üìä UTM Parameter Reference

| Parameter | Purpose | Example |
|-----------|---------|---------|
| `utm_source` | Traffic origin | google, instagram, newsletter |
| `utm_medium` | Marketing channel | cpc, organic_social, email |
| `utm_campaign` | Campaign identifier | brand, services, jan2026_launch |
| `utm_content` | A/B variant | cta_blue, banner_v2 |
| `utm_term` | Paid keywords | private_jet_charter |

## üîß Quick Tips

1. **Consistency**: Use lowercase and underscores (not spaces)
2. **Naming**: Be specific but concise in campaign names
3. **Tracking**: All links auto-track to your Admin ‚Üí Campaigns dashboard
4. **Shortening**: Use bit.ly or similar for long URLs in print/social
5. **Testing**: Always test links before publishing campaigns
